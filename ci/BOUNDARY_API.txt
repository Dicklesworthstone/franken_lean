# BOUNDARY_API.txt — the reviewed public-API allowlist of the unsafe boundary
# crates (D3 law b; bead fln-lld slice 2).
#
# Every bare-`pub` item in a boundary crate MUST have a row here; stale rows
# and undeclared items both fail CI (FLN-STRUCT-022), and the expansion
# covenant (FLN-STRUCT-025) additionally requires the post-expansion public
# item set to be a SUBSET of the rows below — a macro cannot synthesize an
# export. There is no unclassified public item.
#
# Row grammar (six '|'-separated fields, all non-empty):
#   row FLN-BX-NNNN | <path> | <kind> <name> | <surface type> | <evidence> | <no-admission argument>
#
# Shared no-admission ground: fln-unsafe-abi cannot name fln-kernel/fln-checker
# types (FLN-STRUCT-008 prohibited paths + the admission-token tripwire), so no
# output below is expressible as a kernel-checkable term; `Obj` is an opaque
# linear handle whose contents only ever surface as plain data copies.

schema fln-boundary-api/1
